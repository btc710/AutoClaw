# examples/

Reference apps and demo personas built on top of AutoClaw.

Planned examples:

- **hello-copilot/** — the absolute minimum Android Auto / CarPlay app that brings up a copilot, says hello, and ends the conversation when you press the steering-wheel button again.
- **road-dj/** — a music-focused persona that comments on the current track and dances along when parked.
- **road-trip/** — a long-haul co-driver persona that suggests stops, remembers favorites, and plays trivia.

These are reference points, not the canonical UX. Forks and remix personas are encouraged.
