# Code of Conduct

AutoClaw adopts the **Contributor Covenant, version 2.1** as its code of conduct. The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## TL;DR

- Be welcoming, patient, and considerate.
- Disagree respectfully — critique ideas, not people.
- No harassment, no discrimination, no doxxing, no NSFW content in project spaces.
- Assume good faith and ask for clarification before assuming malice.

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, discussions, chat, code reviews, in-person meetups — and when an individual is representing the project publicly.

## Reporting

Please report unacceptable behavior by opening a private security advisory in this repository, or by emailing the project maintainers (contact details will be added once a maintainer team is established). Reports will be reviewed promptly and confidentially.

## Enforcement

Maintainers may warn, temporarily ban, or permanently ban contributors whose behavior violates this Code of Conduct, following the Enforcement Guidelines in the full Contributor Covenant text linked above.
